# AI-Powered Research Paper Summarizer & Insight Extractor

This project is developed as part of the **Infosys Springboard Virtual Internship**.

## 📌 Milestone 1: Data Ingestion & Parsing

### Objective
Build a system to upload research papers in PDF format and extract their textual content for further processing.

### Features Implemented
- Upload research paper PDFs via a web interface
- Backend processing using Django
- Text extraction from digital PDFs using PyPDF2
- Modern and user-friendly frontend UI
- JSON-based API response for extracted content

### Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- PyPDF2
- Git & GitHub

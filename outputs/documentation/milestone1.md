# Milestone 1 – Project Documentation
AI-Powered Research Paper Summarizer & Insight Extractor

## Project Overview
This project focuses on building an AI-powered system to ingest research papers and extract text for further AI-based analysis.

## Objective of Milestone 1
To implement PDF ingestion and text extraction using PyPDF2.

## Scope Covered
- PDF ingestion
- Text extraction from research papers
- Storage of parsed output in JSON format
- Backend-ready project structure

## Implementation
- Developed a PDF parser using PyPDF2
- Extracted text page by page
- Stored output in parsed_text.json
- Validated using sample research paper

## Technologies Used
- Python
- PyPDF2
- JSON
- VS Code

## Output
Successfully generated parsed text in `outputs/parsed_text.json`.

## Status
Milestone 1 completed successfully.
